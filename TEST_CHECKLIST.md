# 웹 대시보드 테스트 체크리스트

## 사전 준비

### 1. 패키지 설치 확인
```bash
pip install -r requirements.txt
```

### 2. 데이터 수집
```bash
python main.py
```

최소한 2일 이상의 데이터가 있어야 비교 기능을 테스트할 수 있습니다.

### 3. 준비 상태 확인
```bash
python check_dashboard_ready.py
```

모든 항목이 ✅로 표시되어야 합니다.
